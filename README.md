# SleepAI — Anàlisi Intel·ligent del Son

Aquest és el repositori de **SleepAI**, una demostració interactiva dissenyada per a la visualització, extracció de coneixement i anàlisi d'informes clínics relacionats amb els trastorns del son.

## 🌐 Demo en línia

Pots provar l'aplicació web directament des del navegador a través del següent enllaç:

👉 **[https://adria-eurecat.github.io/SleepAIdemo/](https://adria-eurecat.github.io/SleepAIdemo/)**

## 🏥 Sobre el projecte

Es tracta d'una interfície *frontend* desenvolupada per mostrar com la intel·ligència artificial pot ajudar a estructurar i interpretar estudis del son (polisomnografies i poligrafies). Les seves funcions principals inclouen:

- Simulació d'extracció de dades mèdiques mitjançant tècniques de NLP.
- Visualització del *Knowledge Graph* (graf de coneixement) de les relacions clíniques.
- Fitxes detallades de pacients amb semàfors de risc i mètriques.
- Tauler de mètriques i *clustering* per visualitzar la població de pacients.

---
*Nota: Totes les dades que es mostren en aquesta aplicació són completament fictícies i es fan servir exclusivament amb finalitats demostratives (Eurecat).*
